The example used for this short tutorial
----------------------------------------

We will consider a fictitious development project `MockProject` with the following tree structure.

~~~
+ MockProject
    + changes
        + 2022
            * 12.txt
        * LICENSE.txt
        * x-todo-x.txt

    + src
        * __init__.py
        * LICENSE.txt
        * mockthis.py
        + tool_config
            * escape.yaml
        * tool_debug.py
        * tool_escape.py

    + tests
        + mockthis
            * escape.yaml
            * test_escape.py

    * pyproject.toml
    * README.md
~~~
