===================================
Liste complète des filtres spéciaux
===================================

info::
    Les filtres sont nommés au format jinjang::``typesimple_action``.


// AUTO - LIST OF SPECIAL FILTERS - START

    * jinjang::``author_name`` est un filtre avec un ¨arg optionnel qui renvoie des ¨infos sur un auteur de type jinjang::``author``.

        1) jinjang::``author_name`` renvoie les prénoms et le nom d'un auteur de type jinjang::``author``.

        1) jinjang::``author_name("last")`` renvoie juste le nom d'un auteur de type jinjang::``author``.

        1) jinjang::``author_name("first")`` renvoie juste les prénoms d'un auteur de type jinjang::``author``.


    * jinjang::``author_email`` renvoie l'adresse du courriel d'un auteur de type jinjang::``author`` (si l'adresse n'a pas été renseignée, une exception sera levée).


    * jinjang::``date_day`` est un filtre avec un ¨arg optionnel pour le format de retour souhaité pour le jour d'une date de type jinjang::``date``.

        1) jinjang::``date_day`` renvoie le numéro à deux chiffres.

        1) jinjang::``date_day("name")`` renvoie le nom dans la langue choisie via le ¨param spécial jinjang::``lang``.

        1) jinjang::``date_day("abrev")`` renvoie l'abréviation du nom dans la langue choisie via le ¨param spécial jinjang::``lang``.


    * jinjang::``date_month`` est un filtre avec un ¨arg optionnel pour le format de retour souhaité pour le mois d'une date de type jinjang::``date``.

        1) jinjang::``date_month`` renvoie le numéro à deux chiffres.

        1) jinjang::``date_month("name")`` renvoie le nom dans la langue choisie via le ¨param spécial jinjang::``lang``.

        1) jinjang::``date_month("abrev")`` renvoie l'abréviation du nom dans la langue choisie via le ¨param spécial jinjang::``lang``.


    * jinjang::``date_year`` est un filtre avec un ¨arg optionnel pour le format de retour souhaité pour l'année d'une date de type jinjang::``date``.

        1) jinjang::``date_year`` renvoie le numéro sur 4 chiffres.

        1) jinjang::``date_year("short")`` renvoie le numéro limité à deux chiffres.

// AUTO - LIST OF SPECIAL FILTERS - END
