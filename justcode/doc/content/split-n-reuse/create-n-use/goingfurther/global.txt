=============================
Indiquer des ¨vals par défaut
=============================

Les ¨vals des ¨params sont cherchées d'abord localement, puis globalement. Par exemple, dans le code suivant, les ¨vals des paramètres peuf::``licence`` sont directement données localement pour chacun des patrons peuf::``changes``, peuf::``readme`` et peuf::``tools``.

peuf::
    ---
    path = ¨exaglob/params-no-global.peuf
    ---


On note ici une répétition de certaines ¨vals ; il serait plus simple d'indiquer directement que les licences pour peuf::``changes`` et peuf::``readme`` utilisent la même ¨val peuf::``by nc sa 4.0``, mais aussi de définir les ¨infos sur l'auteur pour tout le projet.
Pour cela, il suffit de passer via le bloc peuf::``main`` tout en cessant d'utiliser des ¨vals locales via peuf::``changes`` et peuf::``readme`` comme cela est fait dans le code suivant. Noter que la ¨val du ¨param peuf::``licence`` est changée localement pour peuf::``tools``.

peuf::
    ---
    path = ¨exaglob/params.peuf
    ---


note::
    Il est important de retenir que le bloc peuf::``main`` permet de définir des ¨vals par défaut tout en ayant la possibilité de changer ces ¨vals localement.
