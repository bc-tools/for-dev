Transformer la valeur d'un ¨param
=================================

¨jinja propose le concept de "filtre" pour modifier une valeur dans un squelette. Par exemple
((

)),
si python::``name = "WaRRioR"``, alors jinja::``"Hello {{ name | title }}!"`` sera transformé en verb::``Hello Warrior!`` suite à l'utilisation de la fonction "filtre" python::``title`` (qui vient de ¨python directement).
¨justcode permet l'utilisation de filtres "maison" comme nous allons le voir dans cette section.
