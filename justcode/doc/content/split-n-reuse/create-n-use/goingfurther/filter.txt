===============================
Transformer la ¨val d'un ¨param
===============================

¨jinjang gère le concept de "filtre" pour modifier une ¨val dans un squelette. Par exemple, si python::``name = "WaRRioR"``, alors jinjang::``"Hello {{ name | title }}!"`` sera transformé en verb::``Hello Warrior!`` suite à l'utilisation de la fonction "filtre" python::``title`` qui vient de ¨python directement
((
    Cet exemple est des plus mauvais, mais nous allons explorer un cas plus pertinent juste après.
)).
¨justcode permet l'utilisation de filtres "maison" comme nous allons le voir dans cette section.


note::
    L'emploi de tel filtre doit être mûrement réfléchi !
    Une autre approche envisageable est de passer via un validateur : voir cf::``checker.txt``.
