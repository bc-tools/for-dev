=================================
Un ¨1er squelette pour les ¨chgts
=================================

Voici comment définir un squelette paramétrable du dossier des ¨chgts pour les projets ¨cvnum et ¨mathobj
((
    En réalité, ces projets utilisent une version compatible avec le projet ¨tnschges via un squelette proposé par défaut par ¨justcode.
)).

dir-tree::
    ---
    path = examples/templates/changes.txt
    ---


Expliquons ce contenu.

    1) Le fichier path::``x-todo-x.txt`` possède le contenu suivant où jinja2::``{{project_name}}`` est une commande ¨jinja2 qui sera remplacé par le nom du projet à initier.

    jinja2::
        ---
        path = examples/templates/changes/xtodox.txt
        ---

    1) L'absence du fichier path::``LICENSE.txt`` vient de ce que l'ajout d'une licence est indiqué dans le fichier path::``specs.peuf`` du dossier path::``__meta__`` propre au fonctionnement de ¨justcode. Ce fichier admet le contenu suivant.

    peuf::
        ---
        path = examples/templates/changes/specs.peuf
        ---


Il reste à voir comment initier un projet à partir de notre squelette. Nous allons supposer que le chemin absolu du dossier définissant le squelette est path::``mes_squelettes/changes``. Pour initier partiellement le projet ¨cvnum, il faut suivre les étapes suivantes.

    1) On crée un dossier path::``cvnum`` avec le contenu minimal suivant.

    dir-tree::
        ---
        path = examples/templates/cvnum/step-1.txt
        ---

    Le nom du projet est automatiquement celui du dossier, soit ici ``cvnum``, et le fichier path::``about.peuf`` sert à indiquer les paramètres attendus pour spécialiser le squelette. Dans notre cas, ce fichier a le contenu suivant.

    peuf::
        ---
        path = examples/templates/cvnum/about-1.peuf
        ---

    On note ici que le nom du squelette est employé pour indiquer la licence "Creative Commons Attribution - Non Commercial - Share Alike 4.0 International", une licence gérée par défaut par ¨justcode. Il est possible d'indiquer directement un fichier licence stocké soit localement, soit en ligne : voir cf::``?/create/license.txt``.

    1) On lance ¨justcode sur notre dossier path::``cvnum``. Voilà, c'est tout !
