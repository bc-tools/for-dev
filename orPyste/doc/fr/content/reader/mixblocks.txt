=========================
Différents types de blocs
=========================

Voyons comment mélanger différents types de blocs. Ci-dessous, le ¨1er sous-bloc est de type conteneur, non obligatoire ici, le ¨2nd de type clé-valeur, et enfin le ¨3eme de type verbatim ligne par ligne.

orpyste::
    book::
        general::
            auteur = M. Personne
            titre  = Ce livre a-t-il un titre ?
            date   = 1er Mai 2012

        resume::
            Ce livre est une ode au temps qui passe...


En considérant que le chemin complet du fichier est path::``user/example.txt`` avec un encodage en ``utf-8``. L'utilisation de la classe devient la suivante où la nouveauté est l'utilisation de ``container`` pour indiquer des blocs uniquement destinés à contenir d'autres blocs.

python::
    from pathlib import Path

    from orpyste.data import Read

    with ReadBlock(
        content = Path("user/example.peuf"),
        mode    = {
            "container" : ":default:",
            "keyval:: =": "general",
            "verbatim"  : "resume"
        }
    ) as datas:
        print(datas.mydict("std nonb nosep"))


Lancé depuis un terminal, le code nous fournit la sortie suivante (où la ¨mef est "maison").

term::
    {
        'book/general': {
            'auteur': 'M. Personne',
            'date'  : '1er Mai 2012',
            'titre' : 'Ce livre a-t-il un titre ?'
        },
        'book/resume': (
            'Ce livre est une ode au temps qui passe...',
        )
    }
