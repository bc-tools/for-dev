========================
Variables dans un patron
========================

Les variables se tapent via une macro fictive latex::``\JNGVAR{ ... }`` qui devinet active si l'on utilise les outils présentées dans cf::``tools.txt``.

jinjang::
    ---
    flavour = latex
    ---

    J'utilise une variable : \JNGVAR{une_var}.
