===================
Les ¨vars spéciales
===================

Pour faciliter la rédaction des commandes des traitements périphériques, il est possible d'utiliser les ¨vars spéciales suivantes.

    1) jinjang::``{{input}}`` est le nom du fichier "patron", avec son extension.

    1) jinjang::``{{output}}`` est le nom du fichier produit, avec son extension.
