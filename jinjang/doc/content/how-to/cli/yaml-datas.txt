======================================
Avec des données dans un fichier ¨yaml
======================================

La méthode la plus sûre, en terme de sécurité, consiste à travailler avec deux fichiers du type suivant.

    1) Un fichier "patron" que l'on veut peupler.

    1) Un fichier au format ¨yaml associant des valeurs aux variables nécessaires au peuplement du fichier "patron".


Imaginons que nous soyons dans la situation suivante.

    1) Dans un dossier path::``my/folder`` se trouve un fichier "patron" path::``mytemplate.tex`` qui est de type ¨latex.

    1) Les données sont indiquées dans le fichier path::``mydatas.yaml``.


Dans ce cas, pour obtenir un fichier path::``myoutput.tex`` obtenu après peuplement, il suffira de procéder comme suit où l'option term::``-dto`` est pour "[d]atas, [t]emplate, [o]utput", soit  "données, patron, produit" en français (la sortie ¨jinjang a été omise, et d'où la présence de term::``[...]``).

terminal::
    > cd my/folder
    > python -m jinjang -dto mydatas.yaml mytemplate.tex myoutput.tex
    [...]


A titre indicatif, montrons des contenus possibles pour les deux fichiers de départ, et ensuite ce que contient le fichier produit. Commençons par le "patron".

latex::
    ---
    title = Le "patron" path::``mytemplate.tex``
    path  = ?/examples/mytemplate.tex
    ---


Les données suivantes sont utilisées pour peupler le "patron".

yaml::
    ---
    title = Les données dans path::``mydatas.yaml``
    path  = ?/examples/mydatas.yaml
    ---


La commande term::``python -m jinjang -dto...`` fournit finalement le fichier suivant.

latex::
    ---
    title = Le produit fini path::``myoutput.tex``
    path  = ?/examples/myoutput.tex
    ---


note::
    La seule obtention du fichier path::``myoutput.tex`` n'est pas forcément satisfaisante : on pourrait souhaiter obtenir aussi le ¨pdf fabriqué via l'appel de ¨pdflatex appliqué à path::``myoutput.tex``.
    La section cf::``hooks.txt`` explique comment appeler des programmes extérieurs après le travail fait par ¨jinjnag.
